# API Key Configuration Guide

This document provides a comprehensive guide on how to configure API keys for QWeather and Amap for the City Travel Guide project.

## QWeather Setup  
1. Go to the [QWeather official website](https://www.qweather.com/).  
2. Sign up for an account or log in if you already have one.  
3. After logging in, navigate to the API section and select the desired weather API service.  
4. Create a new application to obtain your API key.  
5. Copy your API key and keep it secure.

## Amap Setup  
1. Visit the [Amap developer site](https://developer.amap.com/).  
2. Register for an account and log in.  
3. Once logged in, go to the API management section.  
4. Create a new project to generate your API key.  
5. Copy the generated API key and store it safely.

## Conclusion  
After obtaining both API keys, make sure to integrate them into your application as per your needs. Follow the respective API documentation for further instructions on how to use the keys effectively.
